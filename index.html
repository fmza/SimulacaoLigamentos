O que cada componente do em Computador faz?

Apertar o botão de ligar: Quando o botão de ligar do gabinete é apertado (ligado), ele ativa a Fonte de alimentação

Fonte: a Fonte serve para fornecer energia para os outros componentes do PC, estabilização de tensão e corrente, proteção contra sobrecarga e curto-circuito, mas principalmente ele serve para ativar um programa chamado Boot

Boot: o Boot serve para iniciar o hardware, carregar o firmware, encontrar e carregar o sistema operacional, iniciar o sistema operacional, preparar o ambiente de execução e facilitar a execução de aplicativos

CPU: quando o Boot liga a CPU, a CPU ativa um outro programa chamado POST

POST: o POST serve para a verificação de hardware, detecção de erros, garantia de funcionamento seguro, identificação de dispositivos, configuração de hardware básica e informação do sistema

Ligar os componentes: depois disso tudo é que os componentes do PC realmente ligam e começam a funcionar, depois dos componentes ligarem o sistema operacional do PC começam a ligar

Abrir um aplicativo: depois do PC ligar e depois do sistema operacional iniciar, só aí você irá conseguir abrir um aplicativo
